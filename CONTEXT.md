# Project Context

## What This Is
An interactive quiz web app based on the ALT (AI-First Lean Teams) Playbook. Built as a side project by a designer at the company to learn the playbook content and share it with the team via the design-output channel.

## Why It Exists
The ALT Playbook lives in a Google Doc — useful, but passive. This project transforms that content into an engaging, quiz-based learning experience. The story: "I was trying to learn the playbook, so I built something to help me (and others) test their knowledge."

## Who It's For
- ALT team members and new joiners wanting to learn ALT principles
- Designers and non-technical team members at the company
- Anyone curious about AI-First Lean ways of working

## Tech Stack
- Vanilla HTML, CSS, JavaScript only — no frameworks, no build tools
- No backend — fully static, runs in the browser
- Should be openable by just double-clicking index.html

## Developer Profile
The builder is a designer with HTML and CSS experience. JavaScript should be kept simple, readable, and well-commented. Avoid complex patterns, frameworks, or abstractions.

## Content Source
The ALT Playbook covers 6 Key Strategies:
1. Build to Think
2. Daily Creation of Value
3. Reflexive AI Usage (+ AI as a Translator)
4. Radical Collaboration
5. Everyone Contributes to the End Product
6. Bring the Pain Forward and Continuous Delivery

Each section has principles, practical applications, and "what to watch out for" items — all quiz-worthy content.

## Design Direction
- Clean, modern, professional but approachable
- Should feel like a product, not a school assignment
- Reflects the AI-forward culture of the team
- Mobile-friendly

## Tone
Confident, clear, encouraging. Not corporate-stiff. Matches the ALT Playbook's direct and energetic voice.
